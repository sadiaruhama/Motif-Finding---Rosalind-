# Rosalind – Finding a Motif in DNA (SUBS)

Solution to the **Finding a Motif in DNA (SUBS)** problem from the Rosalind Bioinformatics platform.

## Problem Description

Given two DNA strings `s` and `t`, find all starting positions where `t` appears as a substring of `s`. Positions are reported using **1-based indexing**.

## Topics

- Bioinformatics
- DNA Sequence Analysis
- String Matching
- Pattern Searching
- Python



## How to Run

```bash
python solution.py
```

## Example

### Input

```
GATATATGCATATACTT
ATAT
```

### Output

```
2 4 10
```
